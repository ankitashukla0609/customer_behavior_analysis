# 🛍️ Customer Shopping Behavior Analysis

## 📌 Project Overview

Customer Shopping Behavior Analysis is a data analytics project that examines customer purchasing patterns using transactional data. The project combines Python, PostgreSQL, and Power BI to clean, analyze, and visualize shopping behavior, helping businesses make data-driven decisions.

---

## 🎯 Objectives

- Analyze customer purchasing patterns.
- Understand customer demographics and preferences.
- Identify high-performing products.
- Study subscription and discount behavior.
- Provide actionable business recommendations.

---

## 📊 Dataset Information

- **Total Records:** 3,900
- **Total Features:** 18

### Dataset Includes:

- Customer demographics
  - Age
  - Gender
  - Location
  - Subscription Status

- Purchase information
  - Item Purchased
  - Category
  - Purchase Amount
  - Season
  - Size
  - Color

- Shopping behavior
  - Discount Applied
  - Previous Purchases
  - Purchase Frequency
  - Shipping Type
  - Review Rating

## 📈 SQL Business Analysis

The project answers several business questions:

### 1. Revenue by Gender
Compare total revenue generated by male and female customers.

### 2. High Spending Discount Users
Identify customers using discounts while spending above average.

### 3. Top Rated Products
Find products with the highest average ratings.

### 4. Shipping Type Comparison
Compare spending between shipping methods.

### 5. Subscribers vs Non-Subscribers
Analyze revenue and spending behavior.

### 6. Discount Dependent Products
Identify products frequently purchased with discounts.

### 7. Customer Segmentation
Classify customers into:
- New
- Returning
- Loyal

### 8. Top Products by Category
Find the most purchased products.

### 9. Repeat Buyers and Subscription Trends
Study subscription likelihood among repeat buyers.

### 10. Revenue by Age Group
Analyze revenue contributions across age groups.

## 🚀 How to Run

### Clone the repository

```bash
git clone https://github.com/yourusername/customer-shopping-behavior-analysis.git
```

### Install dependencies

```bash
pip install pandas numpy matplotlib seaborn sqlalchemy psycopg2
```

### Run the Python script

```bash
python data_cleaning.py
```

### Import data into PostgreSQL

Execute the SQL scripts provided.

### Open Power BI Dashboard

Open the `.pbix` file in Power BI Desktop.
